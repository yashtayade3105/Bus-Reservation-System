# 🚌 Bus Reservation System (Java Swing)

A beautiful, beginner-friendly **Java Swing GUI application** for booking bus tickets.  
It supports **login, destination viewing, ticket booking, fare calculation, and bill saving** — all with a clean and gradient-based UI design.

---

## 🚀 Features

✅ **Login System**
- Simple username/password authentication (`admin / 1234`)

✅ **Dynamic Seat Management**
- Each destination has 20 seats initially  
- Seats reduce automatically after every booking

✅ **Ticket Booking**
- Choose source & destination  
- Enter number of passengers  
- Calculates total fare automatically  
- Saves the bill in a `.txt` file

✅ **View Destinations**
- Shows fare and available seats for each destination

✅ **Beautiful GUI**
- Gradient background on login screen  
- Clean buttons with hover effects  
- Built using Java Swing + AWT components

✅ **File Handling**
- Generates and appends booking bills in `BusReservationBill.txt`
